# Spark Tech Academy Website

## Overview
This project is a website for Spark Tech Academy, designed to provide information about online learning for K–12 students. The website features a navigation bar, interactive sections, and responsive design.

## Project Structure
```
sparktech-website
├── src
│   ├── index.html        # Main HTML structure of the website
│   ├── css
│   │   └── style.css     # Styles for the website
│   └── js
│       └── nav-toggle.js  # JavaScript for navigation toggle functionality
├── .gitignore            # Files and directories to ignore by Git
└── README.md             # Documentation for the project
```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.)
- A code editor (optional, for modifications)

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd sparktech-website
   ```

### Running the Website
- Open `src/index.html` in your web browser to view the website.

### Features
- Responsive navigation bar with toggle functionality.
- Interactive sections for courses, resources, and more.

### Usage
- Click the hamburger menu icon to toggle the navigation links on smaller screens.
